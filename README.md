# <p align="center">線上作業批改系統</p>

                  線上批改系統主要是應用本次研究的「程式迴圈結構與語意之分析比較與視覺化機制」和「程式流程結構分析與比較之視覺化機制」。
                  此系統的目的是批改學生上傳作業的CPP檔案，
                  來自動判斷學生程式作業是否跟老師訂定的規則與答案是否一致，
                  也可以透過「程式迴圈結構與語意之分析比較與視覺化機制」所分析的視覺化圖片回饋給學生，
                  幫助學生可以發現自己作業程式碼的錯誤來進行修正，
                  並且達到自動批改學生程式碼的目的。
